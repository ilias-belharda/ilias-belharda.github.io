APT Attack Simulation on Active Directory

Role: Cybersecurity Engineer (Blue Team / SOC)
Context: Simulation of a full APT attack lifecycle on an enterprise Active Directory environment
Tools: Wazuh, Suricata (IPS), Nessus, SQLMap, Meterpreter, AD Hardening
Outcome: Detection rules creation, incident visibility improvement, and security hardening recommendations

Objective

Simulate a realistic APT attack on a virtualized Active Directory infrastructure and implement detection and defense mechanisms.

Attack Phases

  • Reconnaissance & information gathering
  • Spear phishing / initial access vector
  • Exploitation & privilege escalation
  • Persistence
  • Lateral movement & SMB enumeration
  • Detection & response: Wazuh / Suricata (IPS)

Tools

Wazuh, Suricata, Nessus, SQLMap, Meterpreter, Active Directory hardening, password policies, and security configurations.

Outcome

Final report including security recommendations, detection rules, and network hardening measures.
